Hermes

UCF | Senior Design - May, 2025

Hermes is a module designed to facilitate real-time data delivery from a BESS-BMS. It implements a multi-computer grid configuration to process, analyze, compress, and transmit data to a dedicated cloud element. The project consists of three major components: the data simulator, the edge-based data acquisition (eDAQ) matrix, and the cloud element. The data simulator generates predictive data from a BESS-BMS to anticipate system behavior. The eDAQ matrix enables data sampling, analysis, and transmission between its nodes to ensure efficiency. The cloud element leverages architectures that integrate seamlessly with modern industry standards.

Quine-McCluskey

UCF | Computer Aided Design for VLSI - November, 2024

A routine project with the challenge to implement an efficienct Quine-McCluskey (QM) model. QM is an automation technique that allows for logic synthesis at a large scales. This project helped understanding on the topic of logic basis. The Quine-McCluskey algorithm is a method used in digital logic design to minimize Boolean expressions systematically by eliminating redundant terms through a tabular approach. While, by hand, it is the simpler process to understand it does require quite a bit of organization in terms of the boolean values. Even a simple process could lead to several pages of digit elimination between the numbers. While programming lowered the stress of keeping track of the values, keeping track of the logic was a bit of a challenge. Some techniques that allowed for certain outcomes would be detrimental to other configurations such as the differences between pla and bliff inputs.
